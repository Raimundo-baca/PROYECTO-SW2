# PROYECTO-SW2

Este proyecto consiste en el diseño e implementación de una API REST en Node.js para la gestión de eventos académicos universitarios.
La API permite administrar información sobre eventos, ponentes y asistentes, almacenando los datos en MongoDB y ofreciendo operaciones CRUD completas.

El sistema además:
Consume información de una API externa (por ejemplo, datos meteorológicos o geolocalización de ciudades).
Integra respuestas en formato JSON y XML.
Garantiza el funcionamiento incluso si la API externa no está disponible.
Incluye carga automática de datos mediante script npm.
Gestiona una colección con más de 1000 documentos, incluyendo paginación y filtrado.
La arquitectura está basada en una API desacoplada, preparada para ser consumida por cualquier cliente externo.
